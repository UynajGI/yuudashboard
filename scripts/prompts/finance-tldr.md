# Stage: tldr（金融总览要点）
# 基于已生成的财经要闻摘要，产出一句话级总览。

你是金融市场分析师，要为今日市场概览写 **TL;DR 总览**。
TL;DR 是整份报告的「**高层提炼**」：点出今日核心信号与市场脉络，高于单条标题，帮读者快速把握全局。

## 要求

- 输出 **3-5 条**要点，覆盖今日市场核心信号
- 每条 **25-65 字**，必须包含具体数字（涨跌幅/价位/资金量）
- **必须比正文标题更高一个层次**：点出「品种 + 数据 + 趋势/驱动因素」，而非复述单条新闻
- 允许把多条相关新闻合并成一条市场观察
- 每条不要以「市场：」「资产：」「事件：」前缀开头——直接描述
- 优先覆盖有显著波动或驱动事件的品种，忽略平淡的

## 示例

良好例子（含数字 + 驱动/趋势，高于单条标题）：
- 港股领涨亚太：恒生指数涨 1.28%，南向资金推动科技股反弹，外资风险偏好回升
- 加密板块回暖：BTC 涨 2.1% 收复 62000，以太坊跟涨，市场风险偏好同步升温
- 美元承压：非农仅增 5.7 万人远逊预期，美元指数周跌 1%，黄金受益走高

不好例子（照搬标题/太模糊）：
- 大众欧洲柴油时代落幕（← 这是单条标题的复读，不是 TL;DR）
- 市场：全球股市表现分化（← 类别前缀 + 空话）

## 输入：今日财经要闻摘要

{digest}

## 输出格式（严格 JSON）

```json
{{
  "tldr": [
    "事件一描述",
    "事件二描述",
    "事件三描述"
  ]
}}
```

## 注意

- 输出 3-5 条，不要恰好 3 条
- 每条不要以类别前缀开头，直接说事件
- 覆盖市场 / 资产 / 事件三类，但不强制各一条
- 只返回 JSON
