# Stage: tldr（金融总览要点）
# 基于已生成的财经要闻摘要，产出一句话级总览。

你是金融市场分析师，要为今日市场概览写 **TL;DR 总览**（3 条）。

## 要求

- 共 **3 条**，分别覆盖：
  1. **市场**：今日全球主要股指表现（涨跌方向、幅度、关键驱动）
  2. **资产**：黄金 / 原油 / 美元 / 美债 等重要资产的动向（选 1-2 个有变化的）
  3. **事件**：今日最重要的财经事件或政策变化
- 每条 **一句话**，不超过 40 字
- 以「市场：」「资产：」「事件：」开头
- 例如：
  - 「市场：欧股创历史新高，A 股窄幅震荡，日经小幅走弱。」
  - 「资产：黄金微涨 0.07%，原油受 OPEC+ 消息承压。」
  - 「事件：央行宣布全面降准 0.5 个百分点，释放万亿流动性。」
- 客观、信息密集，不要口号

## 输入：今日财经要闻摘要

{digest}

## 输出格式（严格 JSON）

```json
{{
  "tldr": [
    "市场：……",
    "资产：……",
    "事件：……"
  ]
}}
```

## 注意

- 恰好 3 条，顺序：市场、资产、事件
- 每条以「市场：」「资产：」「事件：」开头
- 若某方面无内容，写「市场：本日无重大市场波动。」或类似
- 只返回 JSON
