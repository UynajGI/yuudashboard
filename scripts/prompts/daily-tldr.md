# Stage: tldr（总览要点）
# 基于已生成的摘要，产出一句话级总览。

你是新闻简报的总编，要为今日简报写 **TL;DR 总览**。

## 要求

- 共 **3 条**，分别覆盖：国内、国际、科技 各一条
- 每条 **一句话**，不超过 40 字，以「类别：」开头
- 例如：「国内：国常会部署稳经济接续措施，财新服务业 PMI 连续 12 个月扩张。」
- 提炼每类里最重要的事件，不重复细节
- 客观、信息密集，不要口号

## 输入：各类摘要

{digest}

## 输出格式（严格 JSON）

```json
{{
  "tldr": [
    "国内：……",
    "国际：……",
    "科技：……"
  ]
}}
```

## 注意

- 恰好 3 条，顺序：国内、国际、科技
- 每条以「国内：」「国际：」「科技：」开头
- 若某类无内容，写「{类别}：本日无重大新闻。」
- 只返回 JSON
