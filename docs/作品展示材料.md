# 作品展示材料 —— Yuunagi Dashboard

## 方式一：公网可访问 Demo

### 1. Demo 链接

**https://uynajgi.github.io/yuudashboard/**

站点部署于 GitHub Pages，全球 CDN，HTTPS 加密。评审期间保持可正常访问。

### 2. 测试账号或登录方式

**无需登录。** 站点为完全公开的静态网站，无账号体系、无注册流程、无权限限制。浏览器打开链接即可浏览全部内容。

### 3. 核心功能体验路径

以下路径覆盖系统所有核心功能，预计用时 8-10 分钟。

---

#### 第一步：首页总览（1 分钟）

打开 https://uynajgi.github.io/yuudashboard/

首页以文章卡片瀑布流形式展示所有已生成的日报，最新日报排在最前面。每张卡片包含：

- **分类标签**（时事/金融 + 国内/国际/科技/工程/A股/港股/美股等）
- **标题**（如"时事专栏 · 国内 2026-07-05"）
- **发布时间**
- **TL;DR 今日要点**（蓝色左边框强调，每条 25-60 字，是 LLM 提炼的核心产出）
- **正文预览**

左侧为悬浮分类导航栏，可按分类（时事/金融）和标签快速筛选。

---

#### 第二步：时事专栏 —— 体验 LLM 事件合并与摘要（2 分钟）

点击任意时事专栏（推荐先点"国内"专栏），进入文章详情页：

**页面布局**：
- 顶部：专栏标题 + 发布日期 + 阅读时间
- 右侧：TL;DR 侧边栏（4-6 条今日要点）

**正文结构**（按子类分组，子类标签由 LLM 在 select 阶段自动分配）：

```
### 国内 · 时政
- [**习近平主持中央政治局会议**](原文链接)：会议审议了...（新华社、人民日报、央视新闻）
- [**国务院发布促进消费意见**](原文链接)：明确...（新华社、参考消息）

### 国内 · 经济
- [**央行维持 LPR 不变**](原文链接)：1 年期和 5 年期以上 LPR 分别为...（第一财经、财新网）

### 国内 · 社会
...
```

**AI 体现在哪里**：
- 同一条末尾出现多个来源（如"新华社、人民日报、央视新闻"）—— 这是 LLM 在 select 阶段识别并合并了同一事件的多家报道
- 每条摘要为 1-2 句信息密集的中文，而非 RSS 原文的长篇大论 —— LLM 在 summarize 阶段生成
- TL;DR 要闻比正文标题高一层次（如"俄乌战场出现转折：俄方称完全控制卢甘斯克，同时对基辅发动空袭致 30 死近百伤"）—— LLM 在 tldr 阶段提炼

**建议查看的其他时事专栏**：
- "国际"专栏：体验跨地区新闻的分类和摘要质量
- "科技"专栏：体验橘鸦日报（日更聚合 RSS）经 `splitAggregated` 拆分后的效果
- "工程"专栏：体验对 GitHub Blog、AWS Blog 等技术源的处理

---

#### 第三步：金融专栏 —— 体验数据 + AI 结合（2 分钟）

点击"A 股"专栏卡片，进入金融专栏详情页：

**1. 指数行情表**（纯数据处理，非 AI）：
```
| 指数 | 收盘 | 涨跌幅 |
|------|------|--------|
| 上证综指 | 3,367.23 | +0.12% |
| 深证成指 | 10,826.34 | +0.45% |
| 创业板指 | 2,175.89 | +1.08% |
| 科创50 | 985.67 | +0.73% |
```

**2. chart.xkcd 手绘涨跌柱状图**（纯数据处理 + 前端渲染）：

四根手绘风格柱子，正涨为绿色（`#20bf6b`），下跌为红色（`#eb3b5a`）。视觉上一眼看出哪个指数最强。

**3. 申万行业指数表**（Tushare 数据，按涨跌幅降序排列）：
```
| 行业 | 收盘 | 涨跌幅 |
|------|------|--------|
| 电子 | 3,456.78 | +2.34% |
| 计算机 | 5,678.90 | +1.89% |
| ...（31 个行业完整列表）
```

**4. 北向资金**：`+42.56 亿元`（正数表示外资净流入）

**5. 分类新闻**：
按子类分组（大盘/行业板块/资金流向/个股/政策），格式同时事专栏。

---

#### 第四步：汇总报告 —— 体验 LLM 跨领域综合（2 分钟）

**时事汇总**（"今日要闻汇总"卡片）：

LLM 阅读国内/国际/科技/工程四篇专栏后，自动生成 8-12 条最重要的跨领域要闻。重点观察：LLM 如何将不同专栏的相关内容合并为一条综合要闻（如"OpenAI 与美国政府磋商 + 阿里和解美诉 → AI 治理与中美企业博弈同步升温"）。

**金融汇总**（"金融市场汇总"卡片）：

包含：
1. **LLM 综合点评**（8-12 条）：跨 A 股/港股/亚盘/美股/商品/加密六个市场的综合要闻
2. **全市场指数对比表**：一行对比上证综指、深证成指、恒生指数、日经 225、韩国 KOSPI、道琼斯、纳斯达克、标普 500
3. **全市场涨跌柱状图**：所有指数放在一起的 chart.xkcd 柱状图，一眼看出全球市场涨跌格局
4. **核心资产表**：黄金/WTI 原油/美元指数/BTC/美债 10Y
5. **申万行业涨幅前 10**
6. **各专栏跳转链接**：可跳回单个市场的详细分析

---

#### 第五步：快速验证 AI 加工的完整性（1 分钟）

选任意一条新闻，点击标题链接 → 跳转到原始网站，快速对比：

- 原始 RSS 标题和摘要 vs LLM 生成的中文摘要（是否准确提炼了核心事实）
- LLM 的子类分配是否合理（时政/经济/AI/安全等）
- 多来源标注的条目，是否有至少 2 个来源（验证事件合并确实发生了）

---

### 4. 使用步骤

1. 打开浏览器，访问 **https://uynajgi.github.io/yuudashboard/**
2. 无需登录、注册、安装任何软件或插件
3. 建议使用桌面端浏览器访问（Chrome/Firefox/Safari/Edge 均可）
4. 首次加载约 500ms（含 chart.xkcd 图表 JS），后续页面切换几乎瞬时（Turbo 驱动）
5. 首页向下滚动可浏览历史日报，点击卡片进入详情页
6. 左侧悬浮导航栏可快速切换到时事或金融分类

### 5. 注意事项

1. **站点持续可访问**：部署于 GitHub Pages（全球 CDN），HTTPS 加密，7×24 小时在线
2. **内容每日自动更新**：系统在每日北京时间 19:00 由 GitHub Actions cron 自动触发运行，新日报通常在 19:03-19:08 左右上线。评审期间每天都有新的日报产出
3. **如遇无法访问**：GitHub Pages 全球可用性 > 99.9%，极少发生区域性 DNS 问题。如遇，可尝试刷新页面或使用 HTTPS 直接访问
4. **图表可能不显示**：chart.xkcd JS 库通过 CDN 加载，如 CDN 不可用，页面自动降级为纯 Markdown 表格展示（数字完整可见，不影响信息获取）。此情况极少发生
5. **所有新闻保留原文链接**：每一条摘要都可点击标题跳转到原始来源网站，评委可随时交叉验证 LLM 摘要的质量
6. **移动端可浏览但稍差**：站点以桌面端为主要目标设计，移动端可正常浏览，但 chart.xkcd 图表在窄屏上可能稍小
7. **暗色模式**：站点自适应系统暗色模式设置，无需手动切换

## 附录：AI 能力速查表

评审时可在对应位置观察以下 AI 能力体现：

| AI 能力 | 体现位置 | 如何观察 |
|---------|---------|---------|
| 事件合并（同事件多源整合） | 任何专栏详情页 | 查看摘要末尾的"（来源A、来源B、来源C）"，多来源说明 LLM 合并了多家对同一事件的报道 |
| 中文摘要生成 | 任何专栏详情页 | 每条摘要 1-2 句话，含核心事实和数字，语言自然、无翻译腔 |
| 子类自动分类 | 任何专栏详情页 | 每个 `###` 子标题为 LLM 分配的分类标签，如"国内 · 经济"、"科技 · AI" |
| 要闻提炼（摘要→更高层判断） | TL;DR 侧边栏 | 对比 TL;DR 与正文摘要，TL;DR 含趋势判断和影响分析，正文摘要为客观事实陈述 |
| 跨专栏综合 | 今日要闻汇总页 | LLM 阅读四篇专栏后生成的 8-12 条要点跨领域、按重要性排序 |
| 跨市场综合 | 金融市场汇总页 | LLM 阅读六篇专栏 + 全市场数据后，综合各市场动态生成汇总 |
| Agent 工具调用 | 汇总报告的点评 | LLM 综合中包含具体数字和趋势判断，来自 Agent 对行情和历史的工具调用 |
